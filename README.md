# SeeloewenCraft

SeeloewenCraft is a small 2D sandbox game inspired by Minecraft. You can create and save worlds, break and place blocks to build whatever you like and much more. There's also lot's of stuff planned!
This game is developped as a fun/learn project and is not meant to be publicly available. Note that this game was not created to copy/steal content from Minecraft, just for me to learn C# and WPF a little bit better. All credits of stuff that is inspired by Minecraft go to Mojang.

This repository is only used to store the project and it's builds in a safe place, not to distribute it.
Since this is a fun project, development will not be consistant and may stop entirely at some point.
